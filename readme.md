## Getting Started

### Prerequisites

* Python >= 3.10
* Docker >= 25.0.2
* Docker compose >= v2.27.0
* Make >= 4.3

### Installation


1. Clone the repo
   ```sh
   git clone git@github.com:williamelias/opensearch-tutorial.git
   ```
2. Build images
   ```sh
    $ make build
   ```
3. Start application
   ```sh
   $ make start
   ```
